# Iris Flower Classification using K-Nearest Neighbors (KNN)

## Project Overview

This project demonstrates a complete machine learning classification workflow using Python and Scikit-Learn. It uses the Iris dataset to train a K-Nearest Neighbors (KNN) classifier capable of predicting the species of an iris flower based on its physical measurements.

The project covers the essential stages of a machine learning pipeline, including data loading, preprocessing, model training, prediction, and performance evaluation.

## Dataset

The Iris dataset contains 150 flower samples with four features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target Classes:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

- Scikit-Learn
