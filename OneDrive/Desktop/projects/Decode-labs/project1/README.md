# Enhanced Rule-Based Chatbot

## Overview
This project is an enhanced rule-based chatbot written in Python. It uses simple intent matching and built-in Python libraries to provide useful assistant-like functionality.

## Features

### Conversation
- Greetings (`hello`, `hi`)
- Small talk (`good morning`, `thank you`)
- Help menu
- Status check
- Jokes

### Productivity
- Add tasks
- View task list
- Save notes
- View notes

### Utilities
- Current date
- Current time
- Calculator
- Password generator
- Motivational quotes

### Example Commands

```text
hello
help
status
joke
time
date
calc 25*4
add task Complete assignment
show tasks
note Buy groceries
show notes
motivate
generate password
exit
```

## Requirements

- Python 3.8+

No external dependencies are required.

## Run the Chatbot

```bash
python enhanced_chatbot.py
```
