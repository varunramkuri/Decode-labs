# Project 2: Iris Flower Classification using KNN

## Overview

This project demonstrates a complete machine learning classification workflow using the Iris dataset.

The model predicts the species of a flower based on its measurements using the K-Nearest Neighbors (KNN) algorithm.

## Workflow

1. Load the Iris dataset
2. Split data into training and testing sets
3. Apply feature scaling
4. Train a KNN model
5. Make predictions
6. Evaluate performance

## Technologies Used

- Python
- NumPy
- Scikit-Learn

## Skills Learned

- Supervised learning
- Data preprocessing
- Feature scaling
- Classification algorithms
- Model evaluation
- Confusion matrix analysis



# Project 3: Tech Career Recommendation System

## Overview

This project is a content-based recommendation system that suggests technology career paths based on a user's skills.

The system compares a user's skills with predefined job-role requirements and ranks the most relevant roles using similarity scores.

## Workflow

1. Create a dataset of job roles and skills
2. Build a user profile
3. Convert text into vectors using TF-IDF
4. Calculate similarity using Cosine Similarity
5. Rank job roles
6. Display top recommendations

## Technologies Used

- Python
- Pandas
- Scikit-Learn

## Skills Learned

- Recommendation systems
- Natural Language Processing basics
- TF-IDF vectorization
- Cosine similarity
- Data manipulation using Pandas
